work 2.1 - 表达式求值

输入：
    输入为一个合法的四则运算表达式，不包含括号和任何空白符。
    '+' 与 '-' 只作二元运算符，不以正负号的形式使用。数字均为整数。
    无需对表达式进行校验。

输出： 
    输出表达式的计算结果。

注意：
    1. 所有计算均为整型运算，除法为整数除法 (e.g. 5/2=2)；
    2. 不必考虑除数为 0 的情况。

完成方式：
    参照 expr.h 中的声明在 expr.cpp 中完成 int Eval(const char* const, int) 函数；
    可以在 expr.cpp 中添加自己实现的数据结构或算法。也可删除除 Eval 函数外的其他函数，
    但一定要保证最后结果通过 Eval 函数返回。

    完成后可用 run.bat 进行编译测试。main.cpp 中定义了测试用例，可自行修改测试。

提交方式：

        datastructure2014@yeah.net
